    Para poder descargar automaticamente el dataset debe crearse una cuenta en la página oficial de Kaggle, una vez 
registrado inicie seccion y dirijase a los ajustes de su perfil, ahi podrá solicitar una credencial (token), esto
le descargará un archivo kaggle.json, el cual debe copiar en este directorio. Al hacer esto podrá descargar automaticamene 
el dataset al ejecutar el script download_dataset.py ( Asegúrese de tener instalada la biblioteca kaggle antes de ejecutar el script ).

    En caso de querer hacerlo manual dirijase a "https://www.kaggle.com/code/odins0n/crypto-price-interactive-dashboard-daily-update/output?select=BTCUSD_d.csv" toque en descargar , decomprima el archivo y copie los archivos csv dentro de la carpeta 'db'
