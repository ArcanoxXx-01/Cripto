    Para poder descargar automaticamente el dataset debe crearse una cuenta en la página oficial de Kaggle, una vez 
registrado inicie seccion y dirijase a los ajustes de su perfil, ahi podrá solicitar una credencial (token), esto
le descargara un archivo kaggle.json, el cual debe copiar en este directorio. Al hacer esto podra descargar automaticamene 
el dataset al ejecutar el script download_dataset.py

    En caso de querer hacerlo manual dirijase a " " toque en descargar , decomprima el archivo y copie los archivos csv dentro de la carpeta db/
